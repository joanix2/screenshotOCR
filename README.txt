# Screenshot OCR

The Screenshot OCR application is a powerful tool that simplifies the process of capturing a portion of your screen, performing Optical Character Recognition (OCR) on the captured area, and conveniently copying the resulting text to your clipboard.

## Features

- Capture a selected area of your screen with ease.
- Utilize Optical Character Recognition (OCR) to extract text from the captured image.
- Automatically copy the extracted OCR text to your clipboard.
- Enjoy a user-friendly and intuitive interface for seamless usage.

## Installation

To set up Screenshot OCR on your Linux system, follow these straightforward installation steps:

1. Clone the Screenshot OCR repository to your local computer using the following command:

```bash
git clone https://github.com/joanix2/screeshotOCR.git
```

2. Navigate to the project directory:

```bash
cd screenshotOCR
```

3. Make the installation script executable:

```bash
chmod +x install.sh
```

4. Run the installation script:

```bash
bash install.sh
```

## Usage

Once installed, you can use Screenshot OCR from the command line with the following command:

```bash
screenshotOCR
```

Customize your OCR experience by capturing and extracting text effortlessly with Screenshot OCR.
